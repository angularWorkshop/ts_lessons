# Exercise Template

Replace this file when creating a new exercise branch.

## Goal

Explain in 2-4 lines what the student is expected to learn.

## Task

1. Implement the missing logic in `src/index.ts`.
2. Make all runtime tests pass.
3. Make all type tests pass.

## Constraints

- do not remove exported types unless the task explicitly requires it
- keep strict typing
- avoid `any`

## Run locally

```bash
npm install
npm run check
npm run build
```
