# Exercise 14.2: Getter types

## Goal

Practice key remapping with `as` and template literal types.

## Task

1. Implement a mapped type that prefixes keys with `get`.
2. Capitalize the original key names.
3. Apply it to a service contract.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- use remapping via `as`, not handwritten aliases

## Run locally

```bash
npm install
npm run check
```
