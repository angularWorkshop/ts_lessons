# Exercise 5.3: Index signatures

## Goal

Practice dictionary-like object typing with index signatures, required fields, and a typed fallback lookup.

## Task

1. Describe a translation dictionary with an index signature.
2. Keep required fields next to the dynamic keys.
3. Implement `getTranslation`.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- the dictionary values must stay strings
- fallback must come from the dictionary itself

## Run locally

```bash
npm install
npm run check
```
