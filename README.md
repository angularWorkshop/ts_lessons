# Exercise 6.2: Callback typing

## Goal

Practice typing callback parameters, generic helpers, and predicate function aliases.

## Task

1. Create `Predicate<T>`.
2. Type `mapValues` and `filterValues`.
3. Make callbacks strongly typed.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- preserve generic inference for both helpers

## Run locally

```bash
npm install
npm run check
```
