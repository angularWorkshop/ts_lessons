# Exercise 13.2: Types from functions

## Goal

Practice `ReturnType`, `Parameters`, and `Awaited` by deriving types from real function implementations.

## Task

1. Use `ReturnType` for the result of a factory function.
2. Use `Parameters` for a wrapper around an existing function.
3. Use `Awaited` for the resolved type of an async function.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- avoid duplicating types manually when they can be derived from functions

## Run locally

```bash
npm install
npm run check
```
