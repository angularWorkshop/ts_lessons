# Exercise 7.1: BankAccount

## Goal

Practice classes, access modifiers, readonly fields, and static members.

## Task

1. Implement `BankAccount`.
2. Hide the balance from direct public access.
3. Add a static account counter.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- throw an error when withdrawing more than the current balance

## Run locally

```bash
npm install
npm run check
```
