# Exercise 7.2: Abstract Shape

## Goal

Practice abstract classes, inheritance, and polymorphic work with a shared base type.

## Task

1. Create `abstract class Shape`.
2. Implement `Circle` and `Rectangle`.
3. Implement `sumAreas`.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- `Shape` must not be directly instantiable

## Run locally

```bash
npm install
npm run check
```
