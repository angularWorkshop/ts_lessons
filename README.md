# Exercise 14.1: Manual utility types

## Goal

Practice mapped types by reimplementing common utilities such as `Partial`, `Required`, and `Readonly`.

## Task

1. Implement `MyPartial<T>`, `MyRequired<T>`, and `MyReadonly<T>`.
2. Implement `Nullable<T>`.
3. Implement `DeepPartial<T>`.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- use mapped types rather than handwritten object shapes

## Run locally

```bash
npm install
npm run check
```
