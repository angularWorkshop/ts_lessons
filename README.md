# Exercise 6.1: Overloaded add

## Goal

Practice function overloads, explicit implementation signatures, and typed rest parameters.

## Task

1. Add overloads for `add`.
2. Support `number + number` and `string + string`.
3. Implement `sumAll(...values: number[]): number`.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- do not support mixed `string` and `number` arguments in `add`

## Run locally

```bash
npm install
npm run check
```
