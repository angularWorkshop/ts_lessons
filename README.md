# Exercise 11.2: FunctionReturnType

## Goal

Practice conditional types with `infer` by extracting return values and parameters from real functions.

## Task

1. Implement `MyReturnType<T>` using `infer`.
2. Implement `MyParameters<T>` using `infer`.
3. Apply them to the functions in this exercise.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- utilities must work like the built-in equivalents for valid function types

## Run locally

```bash
npm install
npm run check
```
