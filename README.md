# Exercise 8.2: Discriminated union events

## Goal

Practice discriminated unions, exhaustive switch handling, and `never`-based safety checks.

## Task

1. Model the event union.
2. Fix `assertNever`.
3. Implement `describeEvent` exhaustively.
4. Make all tests pass.

## Constraints

- keep strict typing
- do not use `any`
- exhaustive checking must rely on `never`

## Run locally

```bash
npm install
npm run check
```
